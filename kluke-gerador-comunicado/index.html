👋 Bem-vindo ao Gerador de Comunicados

Esta ferramenta cria comunicados padronizados para os clientes da Kluke, com a identidade visual da empresa já pronta — você só precisa escrever o conteúdo. O resultado pode ser impresso/PDF para uso interno, ou disponibilizado como uma página na internet para o cliente acessar pelo link.

Como funciona

1 Preencha o conteúdo — título, texto de introdução, corpo do comunicado, e adicione listas, alertas ou observações conforme precisar.
2 Acompanhe o preview — o lado direito da tela mostra exatamente como o comunicado vai ficar, em tempo real.
3 Escolha como usar:
  • 📂 Carregar HTML — abre um comunicado já gerado por esta ferramenta para editar e fazer pequenos ajustes, sem precisar digitar tudo de novo.
  • 🖨️ Imprimir — gera uma versão simplificada para imprimir ou salvar como PDF, padronizando os documentos da empresa. Salve o HTML antes, caso queira editar depois.
  • ⬇ Salvar HTML — salva o arquivo do comunicado. Envie para o responsável publicar na internet e compartilhar o link com o cliente por WhatsApp ou e-mail. Dando dois cliques no arquivo salvo, ele já abre direto, sem precisar deste gerador.
  • ↺ Limpar — apaga tudo e começa um novo comunicado do zero.
Dica: pode usar sem medo — nada aqui é salvo automaticamente nem enviado para lugar nenhum, tudo fica só no seu computador. Use o Salvar HTML para guardar o progresso e o Carregar HTML para retomar depois.
